Quick Setup Guide
1
Start the backend
Double-click start-backend.bat — wait for "Uvicorn running on port 8000"
2
Open this dashboard
Double-click dashboard-standalone.html directly in Chrome, or run start-dashboard.bat
3
Add your API key (optional)
Edit backend/.env → replace your_key_here with your real Anthropic or OpenAI key
4
Install Chrome extension
chrome://extensions → Developer Mode ON → Load Unpacked → select extension/ folder
5
Browse and track
Open Gmail, news sites — the extension captures data automatically
